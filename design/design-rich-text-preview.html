Recommended libraries
Rich text engine: Tiptap (vanilla JS, ProseMirror-based, tree-shakable — ideal for no-framework apps)
Emoji picker: emoji-picker-element (~12KB web component, no dependencies, fully CSS-styleable)
GIF search: Tenor API v2 (free, no forced branding — unlike GIPHY free tier)
